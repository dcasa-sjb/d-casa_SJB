## 💻 Como rodar localmente

Basta abrir o `index.html` no navegador, ou servir com qualquer servidor estático:

```bash
# Opção 1: abrir direto
start index.html        # Windows
open index.html         # macOS

# Opção 2: servidor local (recomendado)
python -m http.server 8000
# ou
npx serve .
```

Acesse `http://localhost:8000` no navegador.

## 🌐 Deploy (GitHub Pages)

Este site está pronto para GitHub Pages. Para publicar:

1. Faça push do repositório para o GitHub
2. Vá em **Settings → Pages**
3. Em **Source**, selecione `Deploy from a branch`
4. Escolha a branch `main` e a pasta `/ (root)`
5. O site ficará disponível em `https://<seu-usuario>.github.io/<nome-do-repo>/`
